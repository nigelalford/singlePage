Nigel Alford

Full-Stack Web Developer

I love all things about the internet, web development is a hobby and career. I've loved tinkering on computers since the DOS days. I bring my passion and drive to every project that I touch. My goal is to always leave the application in a better state (visually, and code-wise) than I found it.

I am constantly seeking best practices and methodology improvements in my personal as well as dev life. Along with being a code monkey, I enjoy leading workshops. Teaching others forces me to always search for better ways to work and live so I am in an ever evolving state of learning.
